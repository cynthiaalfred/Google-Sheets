# Google Sheets Project
 Basic Data Cleaning & Feature Engineering
 
# In this project, the following steps were used to clean and prepare the data:

- Grouped data to make it easier to analyse
- Joined cells to combine related information
- Used PROPER to format text correctly (e.g., names)
- Used Find & Replace to fix inconsistent values
- Checked and corrected spelling errors
- Split columns into separate fields where needed
- Extracted date into day, month, and year
- Standardised text using capital case and lower case

### Conditional Formatting
* Created a temperature table with high and low temperatures for each day.
* Selected cells B3:H4 and opened Conditional Formatting.
* Applied colour rules: 30 or more = Orange, 15 or less = Blue,Between 15 and 29 = Yellow
* Formatted the title cell A1 with a green colour.
